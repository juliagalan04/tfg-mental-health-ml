# Mental Health Prediction using Lifestyle Data

This project focuses on developing a machine learning model to predict mental health indicators (stress/work-life balance) based on lifestyle data.

## Datasets
- Lifestyle and Wellbeing Dataset (Kaggle)
- Student Lifestyle Dataset
- Sleep Health Dataset

## Approach
- Data preprocessing (scaling + encoding)
- Classification models:
  - Logistic Regression
  - SVM
  - Random Forest
  - KNN
- Evaluation using Macro F1-score

## Results
Logistic Regression with class balancing achieved the best performance.
